# HAR Viewer

[![license](https://img.shields.io/github/license/develerik/har-viewer.svg)](LICENSE)

Simple viewer for HTTP Archives. All processing is done locally on your device.

## Table of Contents

- [Usage](#usage)
- [Support](#support)
- [Roadmap](#roadmap)
- [Contributing](#contributing)
- [Maintainers](#maintainers)

## Usage

1. Visit [https://develerik.github.io/har-viewer](https://develerik.github.io/har-viewer)
2. Open a `.har` file
   - Option 1: Open a file from your local device
   - Option 2: Open a file from a remote url
3. Analyze your file
